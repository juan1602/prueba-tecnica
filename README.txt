# PRUEBA TÉCNICA - Juan Pablo Ávila Quitián

## 🛠️ Tema personalizado WordPress: "custom-home-2025"

Desarrollado desde cero como parte de la prueba técnica para desarrollador frontend WordPress.

---

## ✅ FUNCIONALIDADES IMPLEMENTADAS

✔️ Tema creado sin constructores ni frameworks  
✔️ Banner dinámico con imágenes desde API externa (https://fakerapi.it)  
✔️ Slider funcional con autoplay, paginación y flechas de navegación  
✔️ Sección de "PILARES" usando Custom Post Type (CPT)  
✔️ Hover en pilares: muestra descripción al pasar el cursor  
✔️ Navbar con logo, menú principal, buscador, idioma e íconos sociales  
✔️ Diseño responsive (desktop y móvil)  
✔️ Menú hamburguesa funcional  
✔️ Código estructurado y documentado  
✔️ Repositorio Git público

---

## 🚀 INSTRUCCIONES DE INSTALACIÓN

1. Clona o descarga este repositorio  
   → https://github.com/juan1602/prueba-tecnica

2. Copia la carpeta `custom-home-2025` en: /wp-content/themes/


3. Activa el tema en WordPress → Apariencia → Temas

4. (Opcional) Importa la base de datos si se incluye archivo `.sql`

5. Desde el panel de WordPress:
- Crea al menos 3 entradas del tipo "Pilar" con imagen destacada
- No necesitas configurar el slider: se carga automáticamente desde la API
